# 💼 Data Science Salary Prediction

An end-to-end Machine Learning project that predicts the salary of data science professionals based on job-related features.
The project includes data preprocessing, exploratory data analysis (EDA), model training, evaluation, and deployment using Streamlit.

# GitHub Repository

# Project Overview

The goal of this project is to build a machine learning model that estimates salaries for data science jobs using information such as:

* Experience Level
* Employment Type
* Job Title
* Employee Residence
* Remote Ratio
* Company Location
* Company Size

The application allows users to input these details and receive an estimated salary prediction through a user-friendly web interface.

# Run Locally

Clone the repository

```bash
git clone https://github.com/Pankaj-Prajapati7521/Salary-Prediction.git
```

Move into the project folder

```bash
cd Salary-Prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
